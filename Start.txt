▶️ START – BugHunter (Guia Rápido)

Guia direto para abrir e rodar o projeto completo.

1️⃣ Clonar o projeto
git clone <URL_DO_REPOSITORIO>
cd bughunter

🐳 2️⃣ Subir tudo com Docker (RECOMENDADO)

Na raiz do projeto:

docker compose up --build


Isso irá:

Subir Backend Flask → http://localhost:5000

Subir Frontend React → http://localhost:3000

Criar network interna entre os serviços

3️⃣ Validar backend

Acesse:

http://localhost:5000/health


Se retornar:

{ "status": "ok" }


Backend funcionando.

4️⃣ Rodar testes automatizados

Com os containers rodando:

pytest -v


Inclui:

Testes unitários

Testes Selenium (E2E do login)

🖥 5️⃣ Alternativa Windows (menu automático)

Execute:

start-docker.bat


ou

start.bat


Para rodar via menu interativo.

🔁 Parar o ambiente
docker compose down

📦 Stack do Projeto

Backend: Flask (Python)

Frontend: React

Testes: Pytest + Selenium

Containerização: Docker + Docker Compose
