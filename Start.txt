
Instruções rápidas

1) Ativar o ambiente virtual (Windows PowerShell):

	.\venv\Scripts\Activate.ps1

2) Rodar os testes:

	pytest -v

	(Se preferir o comando universal: python -m pytest -v)

---

Backend 
Vá para a pasta `backend`, abra o terminal e execute:

	python app.py

Frontend 
Vá para a pasta `frontend`, abra o terminal e execute:

	npm start

```
