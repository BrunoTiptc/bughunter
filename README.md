### ▶️ Como executar o projeto

#### 🔧 Backend

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
pip install -r requirements.txt
python app.py

cd frontend
npm install
npm start

```
